# AI and the Internet

A beginner web development and SEO practice project created using HTML, CSS, GitHub Pages, and Google Search Console.

## Project Overview

This project explores how Artificial Intelligence and the internet are transforming education, productivity, communication, and the future of technology.

The website was built as a practical learning project to understand:
- Web development basics
- HTML and CSS
- Website deployment
- GitHub Pages hosting
- Google Search Console
- SEO and indexing

## Features

- Modern dark-themed UI
- Responsive layout
- SEO meta tags
- Google Search Console verification
- AI and technology content
- GitHub Pages deployment

## Technologies Used

- HTML5
- CSS3
- GitHub Pages
- Google Search Console
